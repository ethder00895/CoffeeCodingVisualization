# CoffeeCodingVisualization

Instructions:

---
1. Clone the repository
```Bash
git clone https://github.com/ethder00895/CoffeeCodingVisualization.git
```

2. Ensure Python 3.12 or lower is installed on your system, then open your command-line interface.

3. Run the following command inside the project directory:
```Bash
pip install -r requirements.txt
```

4. Make sure Node.js and npm are installed. Check your installation with:
```Bash
node -v
```
If the command fails, download Node.js from nodejs.org.

5. Navigate to the project folder. Use your terminal to move into the cloned project directory.
```Bash
npm install -g live-server
```
or
```Bash
npm install -g http-server
```

6. Running either tool will start a local web server and open the site in your browser automatically.
```Bash
http://127.0.0.1:8080
```
Open index.html to view the visualization.

7. Optional: Use Python’s built‑in server. If you prefer, you can serve the project using Python:
```Bash
python -m http.server 8000
```
